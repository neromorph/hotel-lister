# HoteLister
API endpoint untuk aplikasi katalog hotel dengan data berdasarkan negara dan kota dengan fitur review.

# Technology Stack
- Go 1.20.2
- PostgreSQL

# API Documentation
[Postman Documentation](https://documenter.getpostman.com/view/20577005/2s93RTRsZF)

# Setup
Development:
1. go mod tidy
2. go run main.go

Production:
1. go mod download
2. go build
3. ./main
